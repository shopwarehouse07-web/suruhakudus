# suruhakudus
Suruh Akudus adalah penyedia layanan jasa kurir, transportasi, dan titip belanja (Jastip) berbasis lokal yang beroperasi di seluruh wilayah Kabupaten Kudus. Kami hadir sebagai solusi praktis bagi masyarakat yang membutuhkan bantuan untuk mobilitas harian, pengiriman logistik, hingga pemenuhan kebutuhan konsumsi
